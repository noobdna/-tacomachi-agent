# tacomachi-agent

## Name
tacomachi-agent

## Description
Personal SOC and smoke detector for internet-facing systems.

## Overview
tacomachi-agent monitors logs, detects anomalies, and alerts early.
It focuses on early detection rather than prevention.

## Capabilities
- Log monitoring
- Anomaly detection
- Alert notification
- Cloudflare log analysis
- Mail security monitoring
- SSH login monitoring

## Mission
Detect small incidents before they become major incidents.
Act as a smoke detector for servers and internet infrastructure.
